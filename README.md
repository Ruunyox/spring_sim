# Spring Simulation
![results](./results.pdf)
### Modeling a harmonic system with Coulomb Friction
#### Usage
```
>> SPRING SIMULATION

>> PARAMETERS:
     [X_0] - initial starting position
      [dt] - size of time step for euler step
   [STEPS] - how many steps in the simulation

>> RUN AS:
   ./spring_sim [X_0] [dt] [STEPS]
```
